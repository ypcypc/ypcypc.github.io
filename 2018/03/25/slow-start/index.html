Slow Start

emmmm

不錯的日常番吧。

就是搞不清楚為什麼前些陣子有人舉報這部番。搞不懂。

故事講的是女主一之瀨花名因病休學一年，做了一個復讀生，與新的朋友的生活的故事。

不是挺好，挺正能量的麼。舉報啥啊。

唯一一點就是她們交流時偶爾會涉及一些奇奇怪怪的東西。但自信想想交流內容還在正常範圍內啊。難道那些舉報的人就這麼清高平時這麼文雅？

一部優秀的日常番，往往就能融入你的生活。就像之前看點兔一樣。看的很慢，不像看巨人一樣一晚上一季，但這不能說明不喜歡。到完結的時候，還是會傷感。當習慣性的點開b站，發現已經撒花了。

emmmmm

一種說不出來的滋味。
